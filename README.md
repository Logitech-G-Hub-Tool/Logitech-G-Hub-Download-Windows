## 🧠 Overview

**Logitech G Hub** is a central software hub for managing Logitech gaming devices. It allows users to configure lighting effects, set up custom macros, fine-tune performance parameters, and organize profiles for different games. G Hub transforms your peripherals into highly responsive tools, enhancing both competitive and casual gaming by giving you precise control over your hardware.

| Feature | Description |

| 🌐 **Profile Management** | Create and switch between game-specific or application-specific profiles. |
| 🔄 **Cross-Device Sync** | Apply settings across multiple Logitech devices simultaneously. |
